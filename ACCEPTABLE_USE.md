# Acceptable Use

RoAppeal OSS is for legitimate account owners managing their own Roblox appeals.

Do not use this project to:

- Submit dishonest, misleading, or impersonated appeals.
- Spam Roblox Support or any third-party service.
- Evade moderation, bans, rate limits, access controls, or platform rules.
- Use stolen accounts, stolen email inboxes, leaked credentials, or unauthorized proxies.
- Hide abusive automation behind proxy rotation, CAPTCHA solving, or webhook notifications.
- Collect, publish, or trade credentials, session data, proxy credentials, API keys, or private appeal data.

Contributions that make abusive use easier may be rejected even if the code works.

If you operate this tool, you are responsible for complying with Roblox, email provider, proxy provider, CAPTCHA provider, AI provider, Discord, hosting provider, and local legal requirements.
